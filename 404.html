EUAA Monitoring Anonymiser

Upload folders with Word, PDF, text, or Excel files and process them entirely inside the browser. Use either structured anonymisation or PDF black-bar redaction while keeping the original PDF layout.

Privacy posture

  • Client-side processing only
  • No uploads to a custom backend
  • Session data stays in browser memory
  • Optional OCR fallback for scanned PDFs

2. What to anonymise

Recommended for EUAA monitoring demos: Names, case numbers, IDs, addresses, emails, phones, exact dates, locations, facilities, routes, family details, and countries.

Supported: DOCX, PDF, TXT, XLSX. PDFs now support rebuild, preserve-layout replacement, and black-bar redaction. OCR fallback also helps scanned PDFs but may be slower.
